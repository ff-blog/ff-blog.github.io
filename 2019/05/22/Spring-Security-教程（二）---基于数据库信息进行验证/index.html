五、启动运行

至此，基本修改已经完毕，这时候可以启动Application.java启动运行了？答案：不能！！！

【提示】: 本项目的启动类是在app包下，因此我们配置的mapper映射文件的@Repository注解为mybatis的注解，所以boot不会自动注入，因此需要加入@MapperScan(basePackages = “com.bootcap.session.security.mapper”)
